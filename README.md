# CheckOut
front-end web application


CheckOut is a productivity web application built using HTML, CSS, and JavaScript. It helps users manage daily tasks and track daily progress.

## Features

- Add tasks
- Mark tasks as completed
- Delete tasks
- Calendar view
- Streak tracking system
- Progress bar
- Theme switching
- Task suggestions

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download the project
2. Open index.html in a browser
3. Use the app
